kernel of an epimorphism into a free module as finitely generated module

1. Proposition

Let \(R\) be a ring and \(M\) a finitely generated module over \(R\). Suppose \(\pi: M \twoheadrightarrow A^n\) is an epimorphism onto a free module.

Then \(\mathrm{ker}(\pi)\) is finitely generated

2. Proof

direct summand of a finitely generated module as finitely generated module splitting lemma for modules
