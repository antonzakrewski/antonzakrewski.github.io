Traverse Tree

1. Goal

  • go through each Knot

2. Algorithm

  • select root
  • recursion:
    • go through subtree
    • selected not

2.1. Variations:

  • preorder
    • Knot - Left-tree, …, right-tree
  • postorder
    • left-tree, right-tree, knot
  • inorder
    • left-tree, Knot, right-tree
