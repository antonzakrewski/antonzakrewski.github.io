locally path connected space as coproduct of connected components

1. Proposition

Let \(X\) be a locally path connected topological space. Then

\begin{align*} X \cong \coprod_{C_i \mathrm{connected component}} C_i \end{align*}

where \(C_i\) are the connected components

2. Proof

corollary of

  1. homeomorphism and topological sum
  2. connected component as open set in a locally connected set
