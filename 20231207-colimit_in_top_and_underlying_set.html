colimit in Top and underlying Set

1. Proposition

Given category Top and a colimit \(\colim_{I}\), the underlying set is, up to Bijection, the colimit in set.

2. Proof

corollary of:

  • left adjoint preserves colimits
  • forgetful functor from Top to Set admits left adjoints

as

\begin{align*} \mathrm{Set}(\colim_I (A_i)) =& \colim_{I} \mathrm{Set}(A_i) \end{align*}
