covariant ext for a fixed projective module

1. Proposition

Let \(\mathcal{A}\) be an abelian category admitting enough injectives. Suppose \(P \in \mathrm{Ob}(\mathcal{A})\) is a projective object

\begin{align*} \mathrm{Ext}_{\mathcal{A}}^{n}(P,-) = 0 \end{align*}

for \(n \geq 1\)

2. Proof

corollary of:

  • right derived functor of an injective object

where Ext is seen as

\begin{align*} \mathrm{Ext}_{\mathcal{A}}^{n}(-,-): \mathcal{A}^{\mathrm{op}} \times \mathcal{A} \rightarrow \mathrm{Ab} \end{align*}

where \(P \in \mathrm{Ob}(\mathcal{A}^{\mathrm{op}})\) is an injective object
