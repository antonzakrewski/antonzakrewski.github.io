colimit as initial object of a cocone category

1. Proposition

Let \(\mathcal{C}\) be a category, \(J: \mathcal{J} \rightarrow \mathcal{C}\) a diagram and \(\mathrm{Cocone}(\mathcal{J})\) the cocone category. Then a colimit is precisely an initial object in \(\mathrm{Cocone}(\mathcal{J})\)

2. Proof

follows dually to limit as terminal object of a cone category
