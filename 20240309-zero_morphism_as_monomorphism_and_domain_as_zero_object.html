zero morphism as monomorphism and domain as zero object

1. Proposition

Let \(\mathcal{C}\) be a pointed category and \(0: A \rightarrow B\) the zero morphism TFAE:

  1. \(0_{A \rightarrow B}\) is split mono
  2. \(0_{A \rightarrow B}\) is a monomorphism
  3. \(A = 0\) is the zero object

2. Proof

2.1. 1) \(\implies\) 2)

corollary of split mono as monomorphism

2.2. 2) \(\implies\) 3)

Consider the cofork

\begin{displaymath} \begin{tikzcd} A \arrow[shift right=1, swap]{r}{\mathrm{id}} \arrow[shift left=1]{r}{0} & A \arrow[]{r}{0} & B \end{tikzcd} \end{displaymath}

Since \(0\) is a monomorphism, we conclude, that

\begin{align*} \mathrm{id}_{A} = 0 \end{align*}

and thus by zero object and zero morphism as identity it follows, that \(A = 0\)

2.3. 3) \(\implies\) 1)

Note that \(0: 0 \rightarrow B\) is split mono
