G-Set as monoid homomorphism from a group to the hom set

1. Proposition

Let \(G\) be a group, \(X\) a set such that \(g: X \rightarrow X\) is a map for \(g \in G\)

TFAE:

  1. \(X\) is a G-Set
  2. the map
\begin{align*} G \rightarrow& \mathrm{Hom}_{\mathrm{Set}}(M,M) \\ g \mapsto g: X \rightarrow X \end{align*}

is a monoid-homomorphism

see: Hom-Set as Monoid

2. Proof

2.1. 1) \(\implies\) 2)

2.1.1. neutral element

follows from

\begin{align*} e \mapsto& \mathrm{id}_{X} \end{align*}

2.1.2. associativity

follows from definition

2.2. 2) \(\implies\) 1)

2.2.1. neutral element

follows from

\begin{align*} e_G \mapsto& e_{\mathrm{Hom}} = \mathrm{id}_{X} \end{align*}

2.2.2. associativity

follows from monoid homomorphism
