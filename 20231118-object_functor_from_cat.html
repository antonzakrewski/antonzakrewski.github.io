object functor from cat

1. Definition

Given Category Cat, the object funtor is defined

\begin{align*} \mathrm{Ob}: \mathrm{Cat} \rightarrow& \mathrm{Set} \\ \mathcal{C} \mapsto& \mathrm{Ob}(\mathcal{C}) \\ (\mathcal{F}: \mathcal{C} \rightarrow \mathcal{D}) \mapsto& \begin{cases} f: \mathrm{Ob}(\mathcal{C}) \rightarrow& \mathrm{Ob}(\mathcal{D}) \\ x \mapsto& \mathcal{F}(x) \end{cases} \end{align*}
