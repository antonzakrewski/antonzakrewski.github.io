zero object in Set*

1. Proposition

Given Category Set*, the zero object is the singleton \((\{*\},*)\)

2. Proof

2.1. initial

Let \((X,x)\) be a pointed set, then since the base point is preserved

\begin{align*} f: \{*\} \rightarrow& X \\ * \mapsto& * \end{align*}

is the only morphism

2.2. terminal

Let \((X,x)\) be a pointed set, then the only set theoretic map is

\begin{align*} f: X \rightarrow \{*\} x \mapsto * \end{align*}

note that the base point is trivially preserved
