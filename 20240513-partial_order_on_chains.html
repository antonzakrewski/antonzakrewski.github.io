partial order on chains

1. Proposition

Let \(P\) be a partially ordered set, \(\mathrm{Chain}(P)\) the set of chain. Then there exists a natural partial order

\begin{align*} \{p_i\} \leq \{q_j\} :\Leftrightarrow \{p_i\} \subseteq \{q_j\} \end{align*}

2. Proof

follows from

  • inclusion as partially ordered set
  • restriction of a partial order as partial order
