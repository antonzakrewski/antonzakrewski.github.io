finitely generated module over an artinian ring as artinian module

1. Proposition

Let \(A\) be an Artinian ring and \(M\) a finitely generated module over \(A\). Then \(M\) is an Artinian module

2. Proof

2.1. 1) \(\implies\) 2)

Follows from

  • finite sum of artinian modules as artinian module
  • quotient module of an artinian module as artinian module

for

\begin{align*} \pi: \oplus_{i=1}^n A \rightarrow M \end{align*}

2.2. 2) \(\implies\) 1)
