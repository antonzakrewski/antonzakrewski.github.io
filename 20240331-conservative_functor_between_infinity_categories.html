conservative functor between infinity categories

1. Definition

Let \(\mathcal{C}, \mathcal{D}\) be infinity categories and \(\mathcal{F}: \mathcal{C} \rightarrow \mathcal{D}\) a functor between infinity categories Then \(\mathcal{F}\) is said to be conservative, if for all represented simplicial set \(\varphi: \Delta^1 \rightarrow \mathcal{C}\) such that

\begin{align*} \mathcal{F}(\varphi) \end{align*}

is an equivalence between infinity categories then so is \(\varphi\) an equivalence
