cocartesian monoidal category

1. Definition / Proposition

Let \(\mathcal{C}\) be a category with finite coproducts

Then \(\mathcal{C}\) is said to be a cocartesian monoidal category, containing the dual monoidal structure of the cartesian monoidal category

  1. binary coproduct functor as tensoring functor
  2. initial object as tensor unit
  3. tensor functor as induced morphism by the universal property
  4. left unitor / right unitor as coproduct with the initial object canonically iso
