simplicial set and set of simplices

1. Proposition

Let \(X\) be a simplicial set, then there exists a canonical Bijection

\begin{align*} \mathrm{Hom}_{\mathrm{sSet}}(\Delta^n, X) \cong X_n \end{align*}

where \(X_n\) is the Set of n-simplices and \(\Delta^n\) is the represented simplicial set

2. Proof

Yoneda-Lemma By definiton, we get

\begin{align*} \mathrm{Hom}_{\mathrm{sSet}}(\Delta^n, X) = \mathrm{Hom}_{\mathrm{sSet}}(\mathrm{Hom}_{\Delta}(-,[n]), X) \end{align*}

Here an element \(\eta \in \mathrm{Hom}_{\mathrm{sSet}}(\mathrm{Hom}_{\Delta}(-,[n]), X)\) is precisely a natural transformation from \(\mathrm{Hom}_{\Delta}(-, [n])\) to \(X\), hence by contravariant Yoneda it follows, that there exists a natural correspondence between \(\eta\) and elements

\begin{align*} f \in X_n = X(n) \end{align*}
