singular cohomology of the empty set

1. Proposition

Let \(M\) be a group, then singular cohomology of the empty topological space is given by

\begin{align*} H^n(\emptyset, M) = 0 \end{align*}

2. Proof

corollary of:

  • singular cohomology and coproduct
  • terminal object as empty limit
  • \(0\) as terminal object in \(\mathrm{Ab}\)
