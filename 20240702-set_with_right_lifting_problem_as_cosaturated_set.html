set with right lifting problem as cosaturated set

1. Proposition

Let \(\mathcal{C}\) be a category, \(S \subseteq \mathrm{Mor}(\mathcal{C})\) a subcollection of morphisms

\begin{align*} \chi_{R}(S) \end{align*}

the set of morphisms with the right lifting property against \(S\)

Then \(\chi_R(S)\) is a cosaturated set

2. Proof

follows from:

  • left lifting property as right lifting property in the opposite category
  • set with left lifting property as saturated set
