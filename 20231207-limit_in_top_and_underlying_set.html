limit in Top and underlying Set

1. Proposition

Given category Top and a limit \(\lim_{I}\), the underlying set is, up to Bijection, the colimit in set.

2. Proof

corollary of:

  • right adjoint preserves limits
  • limit in Set
  • forgetful functor from Top to Set admits left adjoints
