weakly locally path connected space and sum of path components

1. Proposition

Let \(X\) be a topological space TFAE:

  1. \(X\) is weakly locally path connected space.
  2. \(X\) the topological sum of the path components \(X \simeq \coprod \pi_0([x])\)

2. Proof

corollary of:

  • path and connected components in a weakly locally path connected space
  • weakly locally connceted space and sum of connected components
  • weakly locally path connected space as weakly locally connected space
