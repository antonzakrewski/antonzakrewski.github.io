set with left lifting property as saturated set

1. Proposition

Let \(\mathcal{C}\) be a category, \(S \subseteq \mathrm{Mor}(\mathcal{C})\) a subcollection of morphisms

\begin{align*} \chi_{L}(S) \end{align*}

the set of morphisms with the set with the left lifting property

Then \(\chi_L(S)\) is a saturated set

2. Proof

2.1. a)

set with left lifting property closed under pushouts

2.2. b)

set with left lifting property closed under retracts

2.3. c)

set with left lifting property closed under coproducts

2.4. d)

set with left lifting property closed under sequential colimits
