coproduct with the initial object canonically iso

1. Proposition

Let \(\mathcal{C}\) be a category, \(X \in \mathrm{Ob}(\mathcal{C})\) an object and \(\emptyset \in \mathrm{Ob}(\mathcal{C})\) the initial object Then the coproduct is cannonically isomorphic

\begin{align*} X \coprod \emptyset \cong X \end{align*}
