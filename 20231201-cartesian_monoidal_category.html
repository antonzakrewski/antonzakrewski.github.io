cartesian monoidal category

1. Definition / Proposition

Let \(\mathcal{C}\) be a category with finite products.

Then \(\mathcal{C}\) is said to be a cartesian monoidal category, containing following monoidal structure:

  1. binary product functor as the tensor functor
  2. terminal object as tensor unit (cf. terminal object as empty limit)
  3. associator induced by the universal proprerty
  4. right unitor / left unitor be the product with the terminal object canonically iso
