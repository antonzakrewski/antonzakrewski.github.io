zeroth homotopy group as path components

1. Proposition

Let \((X,x)\) be a based space, then the zeroth homotopy group denotes the path components up to canonical renaming

2. Proof

2.1. construction and continuity

Note that the space of two points \(S^0\) is locally regular compact. Hence the evaluation map from a locally compact as continuous map

\begin{align*} \mathrm{ev}: \mathrm{Map}_{*}(S^0,0), (X,x)) \times S^0 \rightarrow X \end{align*}

is continuous

Therefore, the restriction is also continuous (cf. restriction of a continuous map as continuous map)

\begin{align*} \mathrm{ev}_{1}: \mathrm{Map}_{*}(S^0,0), (X,x)) \times \{1\} \rightarrow X \end{align*}

Furthermore, by product with the terminal object canonically iso, it holds, that

\begin{align*} \mathrm{Map}((S^0,0), (X,x)) \cong \mathrm{Map}((S^0,0),(X,x)) \times \{1\} \end{align*}

Hence we get a continouus map

\begin{align*} \mathrm{Map}_{*}((S^0,0), (X,x)) \rightarrow X \end{align*}

evaluating at the point \(1\)

2.2. bijection

It is a Bijection, as each map \(f \in \mathrm{Map}_*((S^0,0), (X,x))\) is uniquely determined by the non-base point.

2.3. open

Let \(U \subseteq \mathrm{Map}_*((S^1,0), (X,x))\) be open. w.l.o.g. chosing an element of the subbasis, (cf. open injective map and subbasis). Then

\begin{align*} U = \mathcal{O}_{K,O} \end{align*}

we get following cases

2.3.1. \(0 \not\in K\)

if \(U\) is nonempty, then we can choose \(f \in U\) and map \(0\) to an arbitrary point, hence \(U = \mathrm{Map}_*((S^0,0), (X,x))\)

2.3.2. \(x \in U, 1 \in K, 0 \in K\)

Then evaluating results in \(\mathrm{ev}[U] = O\)

2.3.3. \(x \not\in U, 1 \in K, 0 \in K\)

then since our mapping space is pointed, it follows, that \(U = \emptyset\), hence \(\mathrm{ev}[U] = \emptyset\)

2.3.4. \(1 \not\in K, 0 \in K\)

Then evaluating results in \(\mathrm{ev}[U] = O\)

Thus for each case, the image is open.

2.4. equality of path components

by functorality of the path component functor, it follows by isomorphisms under functors, that

\begin{align*} \pi_0(X,x) =& \pi_0(\mathrm{Map}_{*}(S^0,0)) \\ \cong& \pi_0(X) \end{align*}
