closed-weak set and map

1. Proposition

Let \(X\) be a topological space and \(U_i\) a cover of \(X\) such that a set \(A \subseteq X\) is closed if and only if for each \(i \in I\), the intersection \(A \cap U_i\) is closed

Then for a space \(Z\), and a map

\begin{align*} f: X \rightarrow Z \end{align*}

TFAE:

  1. \(f\) is continuous
  2. \(f_i \coloneqq f \circ \iota: U_i \rightarrow Z\) is continuous for each \(i \in I\)

2. Proof

2.1. 1) \(\implies\) 2)

follows from composition of continuous maps and Subspace topology as initial topology for the inclusion

2.2. 2) \(\implies\) 1)

Let \(A \subseteq Z\) be closed, then by assumption, each \(f_i^{-1}[A] \subseteq U_i\) is closed Thus by assumption of \((X,U_i)\) being closed-weak, it follows, that

\begin{align*} f^{-1}[A] \end{align*}

is closed, since \(f_i^{-1}[A] = A \cap U_i\)
