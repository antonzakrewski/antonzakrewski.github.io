projective object in an abelian category

1. Proposition

Let \(\mathcal{A}\) be an abelian category and \(P \in \mathrm{Ob}(\mathcal{A})\)

TFAE:

  1. \(P\) is a projective object
  2. the covariant hom object functor \(\mathrm{Hom}_{\mathcal{A}}(P,-): \mathcal{A} \rightarrow \mathrm{Ab}\) is an exact functor

2. Proof

follows from

  1. projective object and covariant hom functor preserving epimorphisms
  2. epimorphism in Ab as surjection
