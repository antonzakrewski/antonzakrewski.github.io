2. Axiomatische Struktur von \(R\)

  • a) \(W \simeq R^{\mathrm{Spec}_R(W)\)
  • b) \(0 \neq 1\)
  • c) \(x \in R^{\times}\) oder \(1 - x \in R^{\times}\) ("lokal")
  • d) ordnung: \(<\)
    • transitiv
    • \(0 < 1\)
    • \(0 < x, x < 0\) dann ist \(x \in R^{\times}\)
    • \(0< x\) existiert \(y^2 = x\)
    • $ < y → x + z < y + z$
  • e) ordnung \(\leq\)
    • transitiv, reflexiv
    • \(0 \leq 1, \lnot (1 \leq 0)\)
    • \(0 \leq x,y \Rightarrow 0 \leq x \cdot y\)
    • \(x \leq y \Rightarrow x + z \leq y + z\)
    • \(0 \leq d, d \leq 0\) für alle \(d \in D\)
  • natural number object
  • axiom \(\exists n \in N\) \(n > x\)
  • geometrischer körper
\begin{align*} \lnot(x \in \mathrm{Inv}) \rightarrow x = 0 \lnot(x = 0) \rightarrow x \in \mathrm{Inv} \end{align*}
  • pythagoräischer ring,
  • \(\Delta = \{x \vert \forall n \in \mathbb{N} \frac{1}{n} < x < \frac{1}{n} \}\) ist ein maximales ideal
