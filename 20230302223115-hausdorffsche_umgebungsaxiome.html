Hausdorffsche Umgebungsaxiome

1. Satz / Definition

Ein topologischer Raum ist ein Paar \((X, \mathcal{U})\) bestehend aus einer Menge \(X\) und einer Familie \(\mathcal{U} = (\mathcal{U}_x)_{x \in X}\) von Mengen (Umgebungen von \(x\)) von Teilmengen \(u \in \mathcal{U}_x\) , \(u \subseteq X\), so dass gilt:

  1. Jede Umgebung von \(x\) enthält \(x\)
  2. \(X\) ist eine Umgebung für \(x \in X\)
  3. Sei \(U\) eine Umgebung von \(x\) und \(U \subseteq V \subseteq X\), so folgt, dass \(V\) ebenfalls eine Umgebung von \(x\) ist
  4. Der Durchschnitt von zwei Umgebungen eines punktes \(x\) ist wieder eine Umgebung von \(x\) (vgl. Abgeschlossenheit des Umgebunggsystems unter Durchschnitt)
  5. Jede Umgebung von \(x\) enthält eine Umgebung von \(x\), welche Umgebung jedes seiner Punkte ist

2. Beweis

2.1. Topologie \(\Rightarrow\) Umgebungsaxiome

  1. folgt aus der Definition von Umgebung
  2. folgt aus den Mengen in einer Topologie, da \(X\) offen ist
  3. folgt aus der Definition von Umgebung: Es existiert eine offene Menge \(O\) mit \(x \in O \subseteq U\); damit gilt dann auch \(O \subseteq V\)
  4. folgt aus der Abgeschlossenheit des Umgebunggsystems unter Durchschnitt
  5. folgt aus der Offenheit und Umgebung aller Punkte

2.2. Umgebungsaxiome \(\Rightarrow\) Topologie

Sei eine offene Menge definiert als Umgebung aller Punkte in der Umgebung

2.2.1. triviale Mengen in einer Topologie

\(X \in \mathcal{U}\) folgt daraus, dass \(X\) nach Annahme Umgebung aller Punkte \(x \in X\) ist. \(\emptyset \in \mathcal{U}\) folgt nach Definition ebenso.

2.2.2. Vereinigung offener Mengen ist offen

Sei \(U_i\) eine Familie von offenen Mengen nach unserer Definition. Dann ist \(U \coloneqq \bigcup_{i \in I} U_i\) ebenfalls eine offene Menge: Sei \(u \in U\) beliebig, so existiert ein \(i \in I\), so dass \(U_i\) eine Umgebung von \(U\) ist. o.B.d.A. für 2 offenen Mengen, der allgemeine endliche Fall folgt durch Induktion: Seien \(U_1,U_2\) offene Mengen, so gilt für \(U \coloneqq U_1 \cap U_2\), dass \(U\) ebenfall soffen istwa Sei \(u \in U\) beliebig, so sind \(U_1,U_2\) Umgebungen von \(U\) und es folgt, dass \(U\) ebenfalls eine Umgebung ist.
