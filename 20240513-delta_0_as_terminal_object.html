Delta 0 as terminal object

1. Proposition

Let \(\mathrm{sSet}\) be the category sSet and \(\Deta^0\) the represented simplicial set. Then \(\Delta^0\) is the terminal object

2. Proof

follows as

  • covariant hom-functor preserves small limits
  • terminal object as empty limit
  • n-functor preserving objectwise limits and currying
