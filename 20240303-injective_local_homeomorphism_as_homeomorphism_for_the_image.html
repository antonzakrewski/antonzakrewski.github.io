injective local homeomorphism as homeomorphism for the image

1. Proposition

Let \(X,Y\) be topological spaces and \(f: X \rightarrow Y\) a local homeomorphism. Suppose \(f\) is injective, then

\begin{align*} f: X \rightarrow f[X] \end{align*}

is a homeomorphism

2. Proof

for \(y \in f[X]\) there exists an \(x \in X\) with \(f(x) = y\). Since \(f\) is a local homeomorphism, there exists an open set \(O \subseteq X\) such that \(f: O \rightarrow f[O]\) is a homeomorphism, where \(f[O]\) is open. Then \(f^{-1}: f[O] \rightarrow O\) is a homeomorphism.

Thus there exists for each \(y \in f[X]\) a homeomorphism \(g: f[O_y] \rightarrow O_y\). note that they agree on \(f[O_y] \cap f[O_x]\), as \(f\) is injective.

Hence using glueing lemma for open maps on an open covering glueing lemma for an open covering

there exists a continuous, open map

\begin{align*} g: f[X] \rightarrow X \end{align*}

which by construction is an inverse to \(f: X \rightarrow f[X]\)
