locally path connected codomain and covering map

1. Proposition

Let \(B,E\) be topological spaces and \(p: E \rightarrow B\) a covering map. Suppose \(B\) is locally path connected, then so is \(E\)

2. Proof

Let \(e \in E\) and \(O_E \subseteq E\) be an open neighbourhood. Then for \(p(e)\) there exists an open neighbourhood \(U_i\) given by the covering map, such that

\begin{align*} e \in p^{-1}[U_i] \cong F \times U_i \end{align*}

Furthermore, since \(p\) is an open map (cf. covering map as open map), we conclude, that \(p[O_E] \cap U_i\) is open.

Since \(B\) is locally path connected, choose a path connected open neighbourhood \(O_B \subseteq U_i \cap p[O_E]\) of \(p(e)\) Then it follows, that \(e \in \{f\} \times O_B \cong O_B\) (cf. product with the terminal object canonically iso) for some \(f \in F\), so we have our desired path connected neighbourhood
