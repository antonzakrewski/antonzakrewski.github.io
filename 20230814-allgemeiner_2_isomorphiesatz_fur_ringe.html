allgemeiner 2. Isomorphiesatz für Ringe

1. Proposition

Let \(A\) be a commutative ring, \(\mathfrak{a}, \mathfrak{b} \subseteq A\) ideals and \(\pi_{\mathfrak{a}}: A \rightarrow A/\mathfrak{a}\) the canonical epimorphism. Let \(\mathfrak{a}/\mathfrak{b} \coloneqq \pi_{\mathfrak{a}}[\mathfrak{b}]\), then

\begin{align*} (A/\mathfrak{a})/(\mathfrak{b}/\mathfrak{a}) \cong A/(\mathfrak{a} + \mathfrak{b}) \end{align*}

2. Proof

Let

\begin{align*} \varphi: (A/\mathfrak{a})/(\mathfrak{b}/\mathfrak{a}) \rightarrow& A/(\mathfrak{a} + \mathfrak{b}) \\ (a + \mathfrak{a}) + \mathfrak{b}/\mathfrak{a} \mapsto& a + \mathfrak{a} + \mathfrak{b} \end{align*}

2.1. welldefined

Let \((a + \mathfrak{a}) + \mathfrak{b}/\mathfrak{a} = (a' + \mathfrak{a}) + \mathfrak{b}/\mathfrak{a}\) Then there exists an \(b_1 \in \mathfrak{b}\) such that

\begin{align*} (a + \mathfrak{a}) - (a' + \mathfrak{a}) + b_1 =& 0 + \mathfrak{a} \\ ((a - a') + b_1) \in \mathfrak{a} \end{align*}

Therefore, there exists an \(a_1 \in \mathfrak{a}\) such that

\begin{align*} a - a' + b_1 + a_1 = 0 \end{align*}

Hence

\begin{align*} a + \mathfrak{a} + \mathfrak{b} = a' + \mathfrak{a} + \mathfrak{b} \end{align*}

2.2. monomorphism

Using the statement about a minimal kernel and a monomorphism Let \((a + \mathfrak{a}) + \mathfrak{b}/\mathfrak{a} \in \mathrm{ker}(\varphi)\). Then \(a \in \mathfrak{a} + \mathfrak{b}\), hence there exists an \(a' \in \mathfrak{a}\), \(b' \in \mathfrak{b}\) such that

\begin{align*} a + a' + b' = 0 \end{align*}

Therefore

\begin{align*} (a + \mathfrak{a}) + \mathfrak{b}/\mathfrak{a} =& (a + a' + \mathfrak{a}) + (b' + \mathfrak{a}) \mathfrak{b}/\mathfrak{a} \\ =& (a + a' + b' + \mathfrak{a}) + \mathfrak{b}/\mathfrak{a} \\ =& (0 + \mathfrak{a}) + \mathfrak{b}/\mathfrak{a} \end{align*}

2.3. epimorphism

Let \(a + (\mathfrak{a} + \mathfrak{b}) \in A/\mathfrak{a} + \mathfrak{b}\), then we get

\begin{align*} \varphi(a + (\mathfrak{a} + \mathfrak{b})) = a + (\mathfrak{a} + \mathfrak{b}) \end{align*}
