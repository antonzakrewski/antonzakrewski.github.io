set of functions g-set functor as right adjoint

1. Proposition

Let \(G\) be a group, then for category set and category of G-Sets the set of functions g-set functor is a right adjoint to the underlying-set functor.

2. Proof

Let \(X\) be a G-Set and \(Y\) a set.

Then consider

\begin{align*} \varphi: \mathrm{Hom}_{\mathrm{Set}}(\mathrm{Forget}(X), Y) \rightarrow& \mathrm{Hom}_{\mathrm{G-Set}}(X, \mathrm{Hom}_{Set}(G,Y)) \\ (f: x \mapsto y) \mapsto& (x \mapsto (1 \mapsto f(x))) \\ \varphi^{-1}: \mathrm{Hom}_{\mathrm{G-Set}}(X, \mathrm{Hom}_{\mathrm{Set}}(G,Y)) \rightarrow& \mathrm{Hom}_{\mathrm{Set}}(\mathrm{Forget}(X), Y) \\ (x \mapsto (1 \mapsto f_x(1))) \mapsto& (f: x \rightarrow f_x(1)) \end{align*}
