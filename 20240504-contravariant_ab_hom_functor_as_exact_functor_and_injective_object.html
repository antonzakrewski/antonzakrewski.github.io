contravariant ab hom functor as exact functor and injective object

1. Proposition

Let \(\mathcal{A}\) be an abelian category, \(I\) an object and

\begin{align*} \mathrm{Hom}_{\mathcal{A}}^{\mathrm{Ab}}(-,I): \mathcal{A}^{\mathrm{op}} \rightarrow \mathrm{Ab} \end{align*}

the ab-hom functor

TFAE:

  1. \(I\) is an injective object
  2. \(\mathrm{Hom}_{\mathcal{A}}^{\mathrm{Ab}}(-,I)\) is an exact functor
  3. \(\mathrm{Hom}_{\mathcal{A}}^{\mathrm{Ab}}(-,I)\) is a right exact functor

2. Proof

2.1. 1) \(\implies\) 2)

2.2. 2) \(\implies\) 3)

special case

2.3. 3) \(\implies\) 1)
