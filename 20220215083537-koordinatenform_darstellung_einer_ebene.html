Koordinatenform (Darstellung einer Ebene) (Q12)

1. Definition

Eine Darstellung einer Ebene der Form

\begin{align*} E :& a_1 x_1 + a_2 x_2 + a_3 x_3 + c = 0 \end{align*}

ist die Koordinatenform der Ebene
