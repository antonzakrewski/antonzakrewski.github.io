hom set of bimodules

1. Proposition

Let \(R,S,T\) be rings and \(A_1,A_2,B_1,B_2\) bimodules where

  1. \(A_1\) is a \((R,S)\) bimodule
  2. \(A_2\) is a \((S,R)\) bimodule
  3. \(B_1\) is a \((R,T)\) bimodule
  4. \(B_2\) is a \((T,R)\) bimodule

Then the hom-funphism give rise to functors

  1. \(\mathrm{Hom}_{\mathrm{R-Mod}}(A_1,B_1)\) is a \((S,T)\) bimodule
  2. \(\mathrm{Hom}_{\mathrm{Mod-R}}(A_2,B_2)\) is a \((S,T)\) bimodule

with functorality as bifunctor given by precomposition / postcomposition

2. Proof

2.1. a)

see:

  • hom into bimodules bifunctor
  • hom out of bimodules bifunctor

2.2. b)

follows from the flip of bimodules as

  1. \(A_2\) is a \(R^{\mathrm{op}}, S^{\mathrm{op}}\) bimodule
  2. \(B_2\) is a \(R^{\mathrm{op}}, T^{\mathrm{op}}\) bimodule

Therefore

\begin{align*} \mathrm{Hom}_{\mathrm{R^{\mathrm{op}}-Mod}}(A_2,B_2) \end{align*}

is a \((S^{\mathrm{op}}, T^{\mathrm{op}})\) bimodule
