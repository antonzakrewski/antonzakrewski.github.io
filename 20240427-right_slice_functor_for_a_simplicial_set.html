right slice functor for a simplicial set

1. Definition

Let \(S\) be a simplicial set, \(\mathrm{sSet}\) the category sSet and \(S / \mathrm{sSet}\) the coslice category. Then the left slice functor is defined as functor

\begin{align*} -_{p/}: S / \mathrm{sSet} \rightarrow& \mathrm{sSet} \\ (p: S \rightarrow X) \mapsto& \begin{cases} n &\mapsto \mathrm{Hom}_{S/ \mathrm{sSet}}(S \star, \Delta^n \star X ) \\ (f: (p: S \rightarrow X) \rightarrow (p' \rightarrow Y)) &\mapsto f^{\rightarrow}: \mathrm{Hom}_{S / \mathrm{sSet}}(S, \Delta^n \star X) \rightarrow \mathrm{Hom}_{S / \mathrm{sSet}}(S, \Delta^n \star X) \end{cases} \end{align*}
