noetherian space and minimal closed set

1. Proposition

Let \((X, \mathcal{T})\) be a topological space. TFAE:

  1. \(X\) is a noetherian topological space
  2. for each nonempty family of closed subsets, there exists a minimal element

2. Proof(sketch)

2.1. 1) \(\implies\) 2)

follows from Zorn's lemma: Let \(\{C_i\}\) be a set of closed subsets, then each chain stabilizes by assumption. Therefore it also contains a minimal element and hence zorn's lemma applies.

2.2. 2) \(\implies\) 1)

Let $C0 \supseteq C1 \supseteq …$ be a chain. Then take \(\{C_n \vert n \in \mathbb{N}_0\}\). By assumption there exists a minimal element \(C_k\).

Hence this chain stabilzes
