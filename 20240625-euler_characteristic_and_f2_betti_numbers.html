euler characteristic and f2 betti numbers

1. Proposition

Let \(X\) be a homologically finite topological space. Then the homological euler characteristic can be computed by the \(\mathbb{F}_p\)-betti number for a prime number \(p\)

2. Proof

Since \(X\) is homologically finite, we conclude that \(H_n(X)\) is a finitely generated abelian group. Thus using the classification of finitely generated abelian groups it suffices to show this for the cases

\begin{align*} \mathbb{Z} \mathbb{Z}/q^k \end{align*}

for a prime \(q\).

Then using the universal coefficient theorem for singular homology and

\begin{align*} H_n(X,\mathbb{F}_p) = H_n(X) \otimes \mathbb{F}_p \oplus \mathrm{Tor}_{\mathbb{Z}}^{n-1}(H_{n-1}(X,R);\mathbb{F}_p) \end{align*}

Then using additivity of \(\mathrm{Tor}(-,\mathbb{F}_p)\) and the case distiction results in

\begin{align*} \mathrm{Tor}_{\mathbb{Z}}^{1}(\mathbb{Z}/q^k, \mathbb{F}_p) =& \begin{cases} 0 & \mbox{if } (p,q) \mathrm{ coprime} \\ \mathbb{F}_p & \mbox{if } p = q \\ \end{cases} \end{align*}
