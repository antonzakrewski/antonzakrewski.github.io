mapping space from a point homeomorphic to codomain

1. Proposition

Let \(X\) be a topological space, \(*\) the terminal object in category Top. Then the mapping space with the compact open topology

\begin{align*} \mathrm{Map}(*,X) \cong X \end{align*}

is canonically homeomorphic

2. Proof

Let

\begin{align*} f: x \mapsto& (* \mapsto x) \end{align*}

denote the canonical map

Then since \(*\) is compact, it follows that the set

\begin{align*} \{* \mapsto x \vert x \in U\} \end{align*}

is open in the mapping space if and only if \(U\) is open.

Hence \(f\) is a homeomohrpism
