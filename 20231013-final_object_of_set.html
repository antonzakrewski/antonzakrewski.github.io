terminal object in set

1. Proposition

The terminal object of category set is a singleton

2. Proof

Let \(A\) be a set and \(\{*\}\) the singleton.

2.1. Existence

\begin{align*} f: A \rightarrow& \{*\} \\ a \mapsto& * \end{align*}

2.2. Uniqueness

Since \(f: A \rightarrow \{*\}\) is welldefined, for every \(a \in A\) there exists a \(y \in \{*\}\) such that \(f(a) = y\). Since \(\{*\}\) is a singleton, we conclude that \(y = *\) and hence \(f\) is uniqe
