connected component as open set in a locally connected set

1. Proposition

Let \((X, \mathcal{T})\) be a locally connected topological space. Then a connected component \(C \subseteq X\) is open

2. Proof

Let \(c \in C\), then by assumption there exists an open neighbourhood \(U\) such that \(U\) is connected. By Definition of \(C\), we conclude \(U \subseteq C\) and hence \(C\) is open (cf. openness and neighbourhood for all points)
