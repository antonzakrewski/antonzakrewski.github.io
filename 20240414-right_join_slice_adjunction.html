left join slice adjunction

1. Proposition

Let \(\mathrm{sSet}\) be the category sSet, $S ∈ $ and \(S / \mathrm{sSet}\) the coslice category Then the left coslice join functor \(- \star S: \mathrm{sSet} \rightarrow S / \mathrm{sSet}\) is a left adjoint to the left slice functor \(-_{/-}: S / \mathrm{sSet} \rightarrow \mathrm{sSet}\)

2. Proof

2.1. abstract

using

  • left adjoint by dense subcategory
  • presheaf as colimit of representables
  • left coslice join functor preserves colimits

This reduces to show that there exists a natural isomorphism

\begin{align*} \mathrm{Hom}_{S / \mathrm{sSet}}(\Delta^n \star S,-) \overset{!}{\simeq} \mathrm{Hom}_{\mathrm{sSet}}(\Delta^n, -_{/-}) \end{align*}

Let \((p: S \rightarrow X)\) be an arbitrary object Then we get

\begin{align*} \mathrm{Hom}_{S / \mathrm{sSet}}(\Delta^n \star S,(p: S \rightarrow X)) \overset{!}{\simeq} \mathrm{Hom}_{\mathrm{sSet}}(\Delta^n, X_{/p}) \end{align*}

using the covariant Yoneda Lemma an element

\begin{align*} \eta \in \mathrm{Hom}_{\mathrm{sSet}}(\Delta^n, X_{/p}) =& \mathrm{Hom}_{\mathrm{sSet}}(\mathrm{Fun}(\mathrm{Hom}_{\Delta^{\mathrm{op}}}([n],-)), X_{/p}) \end{align*}

is naturally determined by

\begin{align*} X_{/p}([n]) \end{align*}

By definition this evaluates as

\begin{align*} X_{/p}[n] = \mathrm{Hom}_{S / \mathrm{sSet}}(\Delta^n \star S, X) \end{align*}

This shows the natural isomorphis

2.2. concrete

2.2.1. counit

Let \(p: S \rightarrow X\) be an object in \(S / \mathrm{sSet}\).

It remains to construct a natural transformation

\begin{align*} \eta: S \star X_{p/} \rightarrow X \end{align*}

i.e. maps

\begin{align*} \eta(n)s S_n \coprod \mathrm{Hom}_{S / \mathrm{sSet}}(S \star \Delta^n, X) \coprod \coprod_{k + l = n - 1} S_k \times \mathrm{Hom}_{S / \mathrm{sSet}}(S \star -, X) \rightarrow X_n \end{align*}

Using the universal coproduct of the cokernel we define

  1. \(S_n \rightarrow X_n\) by \(p(n)\)
  2. \(\mathrm{Hom}_{S / \mathrm{sSet}}(S \star \Delta^n, X) \rightarrow X_n\) using precomposition with \(\Delta^n \rightarrow S \star \Delta^n\)
  3. \(S_k \times \mathrm{Hom}_{S / \mathrm{sSet}}(S \star \Delta^l, X) \rightarrow X_n\) using precompositino \(\Delta^l \star \Delta^k \rightarrow S \star \Delta^l\)

where both 2) and 3) use the contravariant Yoneda Lemma for morphisms

\begin{align*} \mathrm{Hom}_{S / \mathrm{sSet}}(\Delta^n, X) \rightarrow X_n \\ S_k \rightarrow (\Delta^k \rightarrow S) \end{align*}
