Kriterien einer mock lecture

1. These

Für eine mock lecture sollte man auf folgende Aspekte achten:

  • Einhalten der Zeit => Professionalität, Vorbereitung
  • Didaktik im vordergrund => keine eigene profilierung als Wissenschaftler*in
    • Interaktion
    • Enthusiasmus
    • inhaltliche schwierigkeit
    • Struktur:
      • evtl. wiederholung der kerngedanken "der letzen vorlesung"
  • Originalität
  • Fachliche Sicherheit
