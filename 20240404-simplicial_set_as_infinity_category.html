simplicial set as infinity category

1. Definition

Let \(X\) be a simplicial set. Then \(X\) is said to be an \(\infty\)-category if it has the extension property for all inner horn inclusions

\begin{align*} \Lambda_j^n \rightarrow \Delta^n \end{align*}

for \(n \geq 2, 0 < j < n\)

\begin{displaymath} \begin{tikzcd} \Lambda \arrow[]{d}{} \arrow[]{r}{} & X \\ \Delta_j^n \arrow[dashed]{ru}{\exists}& \end{tikzcd} \end{displaymath}
