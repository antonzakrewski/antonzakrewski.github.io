image of a continuous open map of a locally path connected space

1. Proposition

Let \((X,\mathcal{T}_X)\) and \((Y,\mathcal{T}_Y)\) be topological spaces, \(f: X \rightarrow Y\) a continuous, open map. Suppose \(X\) is locally path connected, then so is the image \(f[X]\) as subspace

2. Proof

Suppose \(y \in f[X]\) and \(O\) an open subset. Then \(f^{-1}[O]\) is open. Choose an \(x \in f^{-1}[y]\) and by locally path connectedness, there exists a path connected neighbourhood \(V \subseteq f^{-1}[O]\). then it follows, that \(f[V]\) is also path connected (cf. image of a pathconnected set as pathconnceted) and still a neighbourhood (cf. open map and neighbourhoods)
