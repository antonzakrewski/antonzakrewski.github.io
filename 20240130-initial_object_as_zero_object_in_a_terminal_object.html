initial object as zero object in a pointed category

1. Proposition

Let \(\mathcal{C}\) be a pointed category and \(\emptyset \in \mathrm{Ob}(\mathcal{C})\) an initial object. Then \(\emptyset \simeq 0\) is canonically isomorphic to the zero object
