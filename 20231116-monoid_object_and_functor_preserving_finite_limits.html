monoid object and functor preserving finite limits

1. Proposition

Let \(\mathcal{C}, \mathcal{D}\) be categories and \(\mathcal{F}: \mathcal{C} \rightarrow \mathcal{D}\) a functor preserving finite limits.

Then for a monoid object \(G \in \mathrm{Ob}(\mathcal{C})\), the \mathcal{F}(G) ∈ \(\mathrm{Ob}(\mathcal{D})\) is also a group object.

2. Proof

corollary of:

  • limit of the empty diagram as terminal object
  • discrete limit as product
  • monoid object and functor preserving finite products and terminal object
