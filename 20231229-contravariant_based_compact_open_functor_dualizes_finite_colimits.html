contravariant based compact open functor dualizes finite coproducts

1. Proposition

Let \((X,x)\) be a based space. Then the contravariant based compact open functor

\begin{align*} \mathrm{Map}_*(-, (X,x)): \mathrm{Top}_* \rightarrow& \mathrm{Top} \\ \end{align*}

functor dualizing finite coproducts to products

2. Proof

2.1. finite, nonempty coproduct

follows from based mapping space and coproduct as homeomorphism

2.2. initial object to terminal object (cf. colimit of the empty diagram as initial object)

Since the initial object in \(\mathrm{top}_*\) is \((*,*)\) (cf. zero object in Top*), it follows, that

\begin{align*} \mathrm{Map}_*((*,*),(X,x)) \end{align*}

is a singleton, because \((*,*)\) is the initial object and thus there only exists one map.

Thus by terminal object in Top, \(\mathrm{Map}_*((*,*),(X,x))\) is the terminal object
