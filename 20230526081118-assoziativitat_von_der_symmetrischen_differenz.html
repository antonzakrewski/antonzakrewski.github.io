Assoziativität der symmetrischen Differenz

1. Satz

Sei \(X\) eine Menge und \(A,B,C \subseteq X\). Dann gilt für die Symmetrische Differenz \(\Delta\)

\begin{align*} (A \Delta B ) \Delta C =& A \Delta (B \Delta C) \end{align*}

2. Beweis

using Symmetrische Differenz als disjunkte Vereinigung we get

\begin{align*} (A \Delta B ) \Delta C =& (A \setminus B \cup B \setminus A) \Delta C \\ =& (A \setminus B) \setminus C \cup (B \setminus A) \setminus C \cup C \setminus (A \setminus B \cup B \setminus A) \\ =& A \setminus (B \cup \setminus C) \cup B \setminus (A \cup C) \cup C \setminus (A \setminus B \cup B \setminus A) \\ % =& … \cup C \setminus (A \setminus B \cup B \setminus A) \\ =& … \cup C \setminus ((A \setminus B) \cup (B \setminus A)) \\ =& … \cup (C \setminus (A \setminus B) \cap C \setminus (B \setminus A)) \\ =& … \cup (C \setminus A \cup (C \cap B)) \cap (C \setminus B \cup (C \cap A)) \\ =& … \cup (C \setminus A \cup (C \cap B)) \cap (C \setminus B \cup (C \cap A)) \\ =& … \cup (C \setminus A \cap C \setminus B) \cup (C \setminus A \cap C \cap A) \cup (C \setminus B \cap C \cap B) \cup (C \cap B \cap C \cap A) \\ =& … \cup (C \setminus (A \cup B) \cup \emptyset \cup \emptyset \cup (C \cap B \cap A) \\ =& … \cup C \setminus (A \cup B) \cup (C \cap B \cap A) \\ =& A \setminus (B \cup \setminus C) \cup B \setminus (A \cup C) \cup C \setminus (A \cup B) \cup (C \cap B \cap A) \\ \end{align*}

where the last term is clearly symmetric.

Here we used:

  1. complement dualizes union and intersection
  2. Distributivgesetz für Vereinigung und Schnitt
  3. complement and brackets
