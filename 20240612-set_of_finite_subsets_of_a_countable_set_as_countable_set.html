set of finite subsets of a countable set as countable set

1. Proposition

Let \(X\) be a countable set. Then the set of finite subset is also countable

2. Proof

follows from

  • Mächtigkeit der Menge der endlichen Teilmengen
  • and the finite case follows from Mächtigkeit der Potenzmenge (endliche Menge)
