opposite simplicial set

1. Definition

Let \(X\) be a simplicial set. Then the opposite simplicial set \(X^{\mathrm{op}}\) is defined as opposite simplicial object

2. Concrete

\begin{align*} X^{\mathrm{op}}: \Delta^{\mathrm{op}} \rightarrow& \mathrm{Set} \\ [m] \mapsto& X[m] \\ (d_l: [m] \rightarrow [m-1]) \mapsto X_{d_{m-l}} \\ (s_l: [m] \rightarrow [m+1]) \mapsto X_{s_{m-l}} \end{align*}
