projective resolution of an object

1. Definition

Let \(\mathcal{A}\) be an additive category, \(A \in \mathrm{Ob}(\mathcal{A})\) an object and \(E_n \rightarrow A\) a resolution. Then \(E_n\) is said to be projective, if each \(E_n\) is a projective object
