initial object in the category of hurewicz cofibrations

1. Proposition

the empty set \(\emptyset\) with the empty map is the initial object in category of Hurewicz cofibrations.

2. Proof

for a topological space \(X\), it follows, that projection

\begin{align*} \mathrm{pr}_1: X \times [0,1] \rightarrow X \times \{0\} \cup \emptyset \times [0,1] = X \times \{0\} \end{align*}

is a retraction. Thus \(\emptyset \rightarrow X\) is a cofibration (cf cofibration and retraction)
