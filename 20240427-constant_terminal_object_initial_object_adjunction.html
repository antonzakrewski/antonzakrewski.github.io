constant terminal object initial object adjunction

1. Proposition

Let \(\mathcal{C}, \mathcal{D}\) be categories, where \(1 \in \mathrm{Ob}(\mathcal{C})\) is the terminal object and \(\emptyset \in \mathrm{Ob}(\mathcal{D})\) is the initial object Then for the constant functors

\begin{align*} \Delta_{1}: \mathcal{D} \rightarrow& \mathcal{C} \\ \Delta_{\emptyset}: \mathcal{C} \rightarrow& \mathcal{D} \end{align*}

there exists an adjunction

\begin{align*} \Delta_1 \dashv \Delta_{\emptyset} \end{align*}

2. Proof

follows from the universal property, as

\begin{align*} \mathrm{Hom}_{\mathcal{C}}(c, \Delta_1(d)) \simeq \{1\} \simeq \mathrm{Hom}_{}(\Delta_{\emptyset}(c), d) \end{align*}

for arbitrary \(c \in \mathrm{Ob}(\mathcal{C}),d \in \mathrm{Ob}(\mathcal{D})\)
