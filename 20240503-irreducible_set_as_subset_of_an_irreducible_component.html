irreducible set as subset of an irreducible component

1. Proposition

Let \((X, \mathcal{T})\) be a topological space, \(A \subseteq X\) an irreducible space. Then there exists an irreducible component \(C\) such that

\begin{align*} A \subseteq C \end{align*}

2. Proof

Consider the set

\begin{align*} S \coloneqq \{S \supseteq A \vert S \textrm{irreducible}\} \end{align*}

Then an irreducible component is precisely a maximal element.

using Zorn's lemma:

2.1. nonempty

follows from \(A \in S\)

2.2. upper bound

Let

\begin{align*} S_0 \subseteq S_1 \subseteq ... \end{align*}

be chain

Then since \(S\) is closed under taking the closure (cf. irreducible subspace and closure) it follows that

\begin{align*} \bigcup S_n \subseteq \bigcup \overline{S_n} \end{align*}

where \(\bigcup \overline{S}_n\) is irreducible by union of a chain of closed irreducible sets as irreducible set

Hence there exists a maximal element.
