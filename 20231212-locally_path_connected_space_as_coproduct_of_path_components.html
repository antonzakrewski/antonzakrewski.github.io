locally path connected space as coproduct of path components

1. Proposition

Let \(X\) be a locally path connected topological space. Then

\begin{align*} X \simeq \coprod_{C_i \mathrm{path component}} C_i \end{align*}

where \(C_i\) are the path components

2. Proof

corollary of:

  1. Pfadkomponente gleich der Wegzusammenhangskomponente in einem lokal wegzusammenhängenden Raum
  2. locally path connected space as coproduct of connected components
