locally path connected space as coproduct of path components

1. Proposition

Let \(X\) be a (weakly) locally path connected topological space. Then

\begin{align*} X \cong \coprod_{C_i \mathrm{path component}} C_i \end{align*}

where \(C_i\) are the path components
