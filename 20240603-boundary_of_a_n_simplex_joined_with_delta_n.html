boundary of a n simplex joined with delta n

1. Proposition

Let \(\Delta^n\) be the represented simplicial set, \(\partial \Delta^m\) the boundary. Then the join is given by

\begin{align*} \partial \Delta^n \star \Delta^m =& \Delta^{n + m + 1} \setminus \{ \textrm{vertices containing } (1,2,...,n) \} \\ \Delta^m \star \partial \Delta^n =& \Delta^{n + m + 1} \setminus \{ \textrm{vertices containing } (m+1,...,m+n+1)\} \end{align*}

2. Proof

2.1. heuristic

\(\partial \Delta^n\) does not contain \((1,...,n)\). Therefore any vertex containing \((1,...,n)\) can not be constructed in

\begin{align*} \coprod (\partial \Delta^n)[n] \times (\Delta^m)[j] = (\partial \Delta^n \star \Delta^m)[n + j] \end{align*}

Each other vertex can be identified with \((\textrm{sides in }\partial \Delta^n) \times (\textrm{sides in } \Delta^m)\)
