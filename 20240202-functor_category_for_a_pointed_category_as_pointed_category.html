functor category for a pointed category as pointed category

1. Proposition

Let \(\mathcal{C}\) be a pointed category and \(\mathcal{J}\) an indexing category. Then the functor category

\begin{align*} \mathrm{Fun}(\mathcal{J},\mathcal{C}) \end{align*}

is also a pointed category with zero object

\begin{align*} 0: \mathcal{J} \rightarrow& \mathcal{C} \\ \end{align*}

2. Proof

corollary of:

  • terminal object as empty limit / colimit of the empty diagram as initial object
  • limit of functors for a complete domain / colimit of functors for a cocomplete codomain

alternatively: hands on approach
