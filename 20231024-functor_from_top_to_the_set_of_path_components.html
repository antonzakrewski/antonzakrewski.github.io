path component functor

1. Proposition

there exists the functor from category Top to category set

\begin{align*} \mathcal{F}: \mathrm{Top} \rightarrow& \mathrm{Set} \\ X \mapsto& \{\pi_0(x)\} \\ (f: X \rightarrow Y) \mapsto& (f': \pi_0(x) \mapsto& \pi_0(f(x))) \\ \end{align*}

2. Proof

2.1. welldefined morphisms

follows from image of a pathconnected set as pathconnceted

2.2. identity

follows from evaluating

\begin{align*} \pi_0(x) = \pi_0(\mathrm{id}(x)) \end{align*}

2.3. composition

follows from evaluating

\begin{align*} \pi_0((g \circ f)(x) ) = \pi_0(g(f(x))) \end{align*}
