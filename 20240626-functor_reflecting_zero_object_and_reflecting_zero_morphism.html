functor reflecting zero object and reflecting zero morphism

1. Proposition

Let \(\mathcal{A}, \mathcal{B}\) be abelian categories and \(\mathcal{F}: \mathcal{A} \rightarrow \mathcal{B}\) be a functor. Supopse \(\mathcal{F}\) reflects any zero morphism

Then \(\mathcal{F}\) reflects the zero object

2. Proof

Let \(0 = \mathcal{F}(A)\). Then it follows that

\begin{align*} \mathcal{F}(\mathrm{id}) = 0 \Rightarrow \mathrm{id} = 0 \end{align*}

and by zero object and zero morphism as identity we are done
