faithfully flat module

1. Definition

Let \(A\) be a commutative ring and \(M\) an \(A\)-module. Then \(M\) is said to be faithfully flat, if for the tensor product it holds for all \(N\)

\begin{align*} M \otimes N = 0 \Leftrightarrow N = 0 \end{align*}
