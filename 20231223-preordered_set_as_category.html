preordered set as category

1. Definition / Proposition

Let \(A\) be a preordered set, then \(A\) is canonically a category with

  1. elements \(\alpha \in A\) as objects
  2. morphism \(f: \alpha \rightarrow \beta\) if and only if \(\alpha \leq \beta\)

2. Proof

2.1. a)

the identity morphism is given by reflexivity

2.2. b)

the composition is given by the transitivity
