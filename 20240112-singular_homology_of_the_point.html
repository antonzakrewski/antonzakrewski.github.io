singular homology of the point

1. Proposition

For the point, the singular homology is

\begin{align*} H_n(*) = \begin{cases} \mathbb{Z} & \mbox{if } n = 0 \\ 0 & \mbox{else } \\ \end{cases} \end{align*}

2. Proof

Note that since \(*\) is the terminal object in Top, \(\mathrm{Hom}_{\mathrm{Top}}(\Delta^n, *) = *\), showing that

\begin{align*} C_{n}^{\mathrm{sing}) = \mathbb{Z}[*] \cong \mathbb{Z} \end{align*}

Then for the boundary map

\begin{align*} d_n = \sum_{i=0}^{n} (-1)^n d_i \end{align*}

with \(d_i\) as the singular boundary map, we conclude that since \(*\) is the terminalobject, it follows, that \(d_i = \mathrm{id}\) and hence

2.1. boundary map: \(n\) even

Then we get

\begin{align*} d_n =& \sum_{i=0}^{n} (-1)^n \mathrm{id} \\ =& 0 \end{align*}

as the zero morphism

2.2. boundary map: \(n\) odd

Then we get

\begin{align*} d_n =& \sum_{i=0}^{n} (-1)^n \mathrm{id} \\ =& \mathrm{id} \end{align*}

as identity morphism

2.3. \(n = 0\)

Since for

\begin{align*} d_0: \mathbb{Z} \rightarrow 0 \end{align*}

the kernel is given by \(\mathbb{Z}\), it remains to show, that \(\mathrm{im}(d_1) = 0\).

2.4. \(n \geq 1\)

2.4.1. \(n\) even

Then it follows, that

\begin{align*} \mathrm{ker}(d_{n})/\mathrm{im}(d_{n+1}) \cong& 0/0 \\ \cong 0 \end{align*}

2.4.2. \(n\) odd

Then it follows, that

\begin{align*} \mathrm{ker}(d_{n})/\mathrm{im}(d_{n+1}) \cong& \mathbb{Z}/\mathbb{Z} \\ \cong 0 \end{align*}
