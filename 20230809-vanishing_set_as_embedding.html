Vanishing set as Injection

1. Proposition

Let \(A\) be a commutative ring, \(\mathrm{Spec}(A)\) the Zariski spectrum and \(\mathfrak{p}, \mathfrak{q} \in \mathrm{Spec}(A)\) prime ideal TFAE:

  1. \(\mathfrak{p} = \mathfrak{q}\)
  2. \(V(\mathfrak{p}) = V(\mathfrak{q})\)

2. Proof

2.1. 1) \(\implies\) 2)

the vanishing set is well defined

2.2. 2) \(\implies\) 1)

Since \(\mathfrak{p} \in V(\mathfrak{p})\) we get \(\mathfrak{p} \in V(\mathfrak{q})\) or \(\mathfrak{p} \subseteq \mathfrak{q}\). By reasons of symmetry we get \(\mathfrak{p} = \mathfrak{q}\).
