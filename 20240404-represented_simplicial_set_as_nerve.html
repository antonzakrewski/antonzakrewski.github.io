represented simplicial set as nerve

1. Proposition

Let \(n \in \mathbb{N}\) and \(\Delta^n\) be the represented simplicial set. Then for the nerve there exists the isomorphism

\begin{align*} \Delta^n \simeq N([n]) \end{align*}

where \([n] = \{0,...,n\}\) is the partially ordered set as category

2. Proof

using natural isomorphism and isomorphism on the objects it suffices to show, that there exists a natural transformation which is an iso on objects Using the definitions gives

\begin{align*} (\Delta^n)_m =& \mathrm{Hom}_{\Delta}([m],[n]) \\ N([n])_m =& \mathrm{Fun}([m],[n]) \end{align*}

Using the functor from poset to cat as fully faithful functor gives an isomorphism

\begin{align*} \mathrm{Fun}([m],[n]) = \mathrm{Hom}_{\mathrm{Cat}}([m],[n]) \simeq \mathrm{Hom}_{\Delta}([m],[n]) \end{align*}
