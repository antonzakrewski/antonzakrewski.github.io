Org Drill

1. What Problem is it solving

  • Spaced Repetition -> learn facts

2. Usage

2.1. Write

  • write headline
  • add ':drill:'
  • write below first headline:
    • question
  • below there -> heading answer + e.g. notes

2.1.1. Special Syntax

2.1.1.1. Empty
  • if Empty data (excluding property data) -> not for review
    • use as skeleton
2.1.1.2. Cloze
  • omit answer as special heading
  • put answer in cloze
  1. Basic

    [answer]

  2. Hint

    [hint|answer]

2.1.2. Special Card Types

2.1.2.1. 2 Sided
  • a \(\Leftrightarrow\) b
  • randomly choose a or b, add it as question; opposite becomes answer
  • 3. … headlines only as answer

:Properties: :DRILL_CARD_TYPE: twosided :END:

2.1.2.2. multisided
  • 3. … headlines as possible question

:Properties: :DRILL_CARD_TYPE: multisided :END:

2.1.2.3. Multi-Clozed

:Properties: :DRILL_CARD_TYPE: type = :END:

  1. Types
    • hide1cloze
    • hide2cloze
    • show1cloze
    • show2cloze
    • hide1firstmore
      • 75% -> hide first gap
    • show1firstless
      • 75% -> show first gap
    • show1lastmore
      • 75% -> show last gap

2.2. Review

  • M-x org-drill
  • press any key to show answer
  • rate 0-5
    • 0-2 forgot (0 worst, 2 least worst)
    • 3-5 known (3 tough, 5 excellent)

2.2.1. Cram -> review all

M-x org-drill-cram
